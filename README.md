## What is AirSight?

AirSight is an end-to-end **computer vision + RAG system** that estimates air pollution levels directly from aerial or street-level photographs — no sensors, no location data, no prior readings required.

Point it at an image of the sky or city skyline and it returns predicted concentrations for **7 pollutants** (PM2.5, PM10, O₃, CO, SO₂, NO₂, AQI), an **AQI health category**, and a **citation-backed health advisory** generated by Gemini 2.5 Flash grounded in official EPA and CPCB guidelines.

## Model Performance

Evaluated on **1,061 held-out test images**. All 7 pollutants exceed R² = 0.92.

| AQI | **0.9634** | 14.72 | — |
| CO | **0.9539** | 17.33 | ppm |
| PM2.5 | **0.9504** | 20.45 | µg/m³ |
| PM10 | **0.9475** | 16.50 | µg/m³ |
| SO₂ | **0.9322** | 1.75 | ppb |
| NO₂ | **0.9204** | 5.54 | ppb |
| **Overall** | **0.9475** | — | — |

> AQI MAE of 14.72 points means the model is on average **less than one AQI category boundary** (50 pts) away from ground truth.

## Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                     User's Browser                              │
│              https://airsight.vercel.app                        │
│                    React + Vite + Tailwind                      │
└──────────────┬──────────────┬───────────────────────────────────┘
               │              │
       POST /predict    POST /analyze   POST /chat
               │              │              │
               ▼              ▼              ▼
┌─────────────────────────────────────────────────────────────────┐
│              FastAPI Backend (HF Spaces, 16 GB RAM)             │
│           https://gh0stfreak-airsight.hf.space                  │
│                                                                 │
│  ┌────────────────┐  ┌─────────────────┐  ┌─────────────────┐  │
│  │  ResNet-50 CNN │  │  FAISS + MiniLM │  │  Gemini 2.5     │  │
│  │  7-head multi- │  │  all-MiniLM-L6  │  │  Flash (API)    │  │
│  │  output regr.  │  │  vectorstore    │  │                 │  │
│  └────────────────┘  └─────────────────┘  └─────────────────┘  │
│         ↑                    ↑                      ↑           │
│   best_model.keras    artifacts/vectorstore   GEMINI_API_KEY    │
│   (168 MB, HF Hub)    (HF Hub, private)       (HF Secret)      │
└─────────────────────────────────────────────────────────────────┘
```

**Pipeline per request:**

1. Image → ResNet-50 → 7 normalized predictions
2. `scaler.pkl` inverse-transforms → real-world units (µg/m³, ppb, ppm)
3. FAISS similarity search → top-3 relevant EPA/CPCB PDF chunks
4. Gemini 2.5 Flash → health advisory grounded strictly in retrieved chunks
5. Response → frontend renders AQI gauge, pollutant cards, advisory text

## Tech Stack

| Layer | Technology |
|---|---|
| **Vision Model** | ResNet-50 (ImageNet pretrained) + 7 regression heads, trained with mixed-precision float16 |
| **Training** | TensorFlow 2.16, 2-phase (frozen backbone → full fine-tune), GPU via WSL |
| **RAG** | LangChain + FAISS + `all-MiniLM-L6-v2` sentence embeddings (CPU) |
| **LLM** | Google Gemini 2.5 Flash via `google-generativeai` SDK |
| **Backend** | FastAPI + Uvicorn, `slowapi` rate limiting |
| **Frontend** | React 18 + Vite + Tailwind CSS v3 |
| **Backend Hosting** | Hugging Face Spaces (Docker, free 16 GB RAM) |
| **Frontend Hosting** | Vercel (hobby plan, free) |
| **Artifacts** | Hosted on private HF model repo, downloaded at container startup |

## API Reference

Base URL: `https://gh0stfreak-airsight.hf.space`

Interactive docs: [`/docs`](https://gh0stfreak-airsight.hf.space/docs) (Swagger UI)

### `GET /health`
Liveness check. Returns model and vectorstore load status.

```json
{"status": "ok", "model_loaded": true, "vectorstore_loaded": true}
```

### `POST /predict`
CNN inference only. Fast (~2–3s). Rate limited: **10 req/min per IP**.

- **Input:** `multipart/form-data` with `file` field (JPG/PNG/WebP)
- **Output:** AQI score, category, all 7 pollutant predictions with units

### `POST /analyze`
Full pipeline: CNN + FAISS retrieval + Gemini advisory. (~5–10s). Rate limited: **5 req/min per IP**.

- **Input:** Same as `/predict`
- **Output:** Everything from `/predict` + `advisory` text + `sources` list

### `POST /chat`
RAG-grounded Q&A. Rate limited: **5 req/min per IP**.

```json
{
  "question": "Should I wear a mask today?",
  "predictions": {
    "PM2.5": 167.52, "PM10": 197.82, "O3": 68.11,
    "CO": 37.40, "SO2": 8.67, "NO2": 57.47, "AQI": 186.36
  }
}
```
`predictions` is optional — include it for context-aware advice.

## RAG Knowledge Base

The health advisory and chat endpoints are grounded in **5 official air quality documents** (not in the repo — used to build the FAISS vectorstore):

| Document | Source |
|---|---|
| WHO Air Quality Guidelines 2021 | World Health Organization |
| AQI Technical Assistance Document | US EPA |
| About AQI | US EPA |
| Health Effects of Particulate Matter | US EPA |
| Ambient (Outdoor) Air Pollution | WHO |

The vectorstore was built using `all-MiniLM-L6-v2` sentence embeddings (~700-char chunks). Gemini is instructed to answer **only** from retrieved chunks — no hallucination.

## Local Development

### Prerequisites
- WSL (Ubuntu) with Python 3.12 virtualenv at `~/airsight/linux_env`
- Node.js (for frontend)
- `.env` file at project root: `GEMINI_API_KEY=your_key`
- `artifacts/` directory with trained model (not in repo — download from HF Hub or train locally)

### Run Backend (WSL)

```bash
wsl
cd /mnt/c/Users/Meraj/Desktop/PROJECTS/airsight
source ~/airsight/linux_env/bin/activate
uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload
# → http://localhost:8000/docs
```

### Run Frontend (Windows)

```bash
cd frontend
npm install
npm run dev
# → http://localhost:5173
```

### Run Full Pipeline (CLI)

```bash
# WSL, venv active
python run_pipeline.py --image "path/to/image.jpg"

# Quick mode (no FAISS, faster)
python run_pipeline.py --image "path/to/image.jpg" --mode quick
```

### Evaluate Model Accuracy

```bash
cd ~/airsight
source linux_env/bin/activate
python evaluate.py
# Outputs: artifacts/evaluation_report.csv, artifacts/metrics_summary.csv
```

## Known Limitations

| Limitation | Notes |
|---|---|
| Cold start ~60–90s | HF Space sleeps after 30 min inactivity; first visitor triggers model download + TF load |
| CPU-only inference | ~2–5s per image; no GPU on free HF tier |
| Gemini free quota | 500 req/day, 10 req/min — `slowapi` rate limiting (5/min) protects this |
| Image-only input | No sensor fusion, no GPS, no temporal context (yet) |
| India-centric training | Dataset is Mumbai/India focused — may generalize less well to other regions |
